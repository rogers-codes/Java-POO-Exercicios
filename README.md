# Java - Exercícios de Programação Orientada a Objetos

Repositório com exercícios práticos desenvolvidos em Java para consolidar fundamentos de Programação Orientada a Objetos.

## 📂 Estrutura

Os projetos estão organizados dentro da pasta:

POO/src

Cada diretório representa um exercício independente com sua própria classe `Principal` para execução.

## 🧠 Conceitos aplicados

- Criação e modelagem de classes
- Encapsulamento
- Uso de listas (ArrayList)
- Métodos e lógica de negócio
- Organização em pacotes

## ▶ Como executar

1. Clone o repositório
2. Abra na IDE de sua preferência (Para Java, o recomendado sera o InteliJ)
3. Execute a classe `Principal` do projeto desejado
